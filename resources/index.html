Proyecto de Sistemas Operativos - Concurrencia y Rendimiento

✅Estado del Servidor

El servidor HTTP está funcionando correctamente y puede manejar múltiples tipos de concurrencia:

  • 🔄 FIFO - Un cliente a la vez
  • 🧵 Threaded - Hilos bajo demanda
  • 👷 Pre-Threaded - Pool de hilos
  • 🧬 Forked - Procesos bajo demanda
  • 👨‍👧‍👦 Pre-Forked - Pool de procesos

📁Archivos de Prueba

🌐test.html - Página de prueba completa 📄sample.txt - Archivo de texto plano ❌Probar página 404 (archivo inexistente)

📊Cómo probar el servidor

Puedes usar el cliente de benchmark incluido:

./client 10 5 test.html [IP_SERVIDOR] 8080

Esto ejecutará 5 hilos, cada uno haciendo 10 requests al archivo test.html
